---
title: "YAGNI (You Ain't Gonna Need It)"
order: 4
difficulty: "Beginner"
tags: ["code-quality", "agile", "best-practices"]
---

Never add functionality you think you might need in the future. Implement things when you actually need them.

## Why It Matters

- **Wasted time** — building features nobody uses
- **Code bloat** — unused code needs maintenance
- **Complexity** — unused abstractions confuse readers
