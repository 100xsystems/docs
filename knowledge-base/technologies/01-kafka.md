Distributed event streaming platform for high-performance data pipelines and streaming analytics.

## Key Concepts

- **Topic** — Category/feed for published records
- **Partition** — Sub-division for parallel processing
- **Producer** — Publishes records to topics
- **Consumer** — Subscribes and processes records
- **Broker** — Server that stores and serves data
