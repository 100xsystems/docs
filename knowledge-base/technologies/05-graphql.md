Query language for APIs and runtime for executing queries with existing data.

## Key Concepts

- **Schema** — Types and relationships
- **Query** — Read data
- **Mutation** — Write data
- **Subscription** — Real-time via WebSockets
- **Resolver** — Returns data for a field
