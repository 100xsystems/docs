Distributed version control system that tracks changes in source code.

## Key Concepts

- **Repository** — Collection of files and history
- **Commit** — Snapshot of changes
- **Branch** — Independent development line
- **Merge** — Combining changes from branches
- **Pull Request** — Proposed change review
