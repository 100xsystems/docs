# Claude Code — AI-Powered Coding Agent

Build an AI-powered coding agent that operates directly in your terminal.

- **Repository:** [100xsystems/claude-code](https://github.com/100xsystems/claude-code)
- **Difficulty:** Advanced
- **Estimated Time:** 20 hours

## Tracks

| Track | Language | Status |
|-------|----------|--------|
| TypeScript | TypeScript | ✅ 7 lessons across 3 modules |
| Java | Java | 🚧 2 lessons in module 1 |

## Modules (TypeScript Track)

### Module 1: CLI Foundations (4 lessons + quiz + challenge)
1. Introduction & Project Setup
2. Build the CLI Tool
3. Agent Loop (Think → Act → Observe)
4. Implement Tools

### Module 2: AI Integration (2 lessons)
1. LLM Integration & Streaming
2. Context Management

### Module 3: Tools & Plugins (1 lesson)
1. Tool System & Plugin Architecture

## What You'll Build

A fully functional AI coding agent with:
- Interactive CLI with chat, execute, and pipe modes
- Agent loop for reasoning and action
- Plugin/tool system for extensibility
- Streaming LLM responses
- Context window management
- File I/O, command execution, and search tools

## Validation

Each lesson has behavioral tests in `tests/behavior.test.ts` that verify your implementation actually works by importing your compiled modules and asserting behavior.
